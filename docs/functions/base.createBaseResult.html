      • ok: <T>(value: T) => Ok<T>

        Creates a successful Result containing the given value.

        Example

        const result = ok("hello");
        console.log(result); // { type: "Ok", value: "hello" }
      • err: <E>(error: E) => Err<E>

        Creates an error Result containing the given error.

        Example

        const result = err("Something went wrong");
        console.log(result); // { type: "Err", error: "Something went wrong" }
      • isOk: <T, E>(result: Result<T, E>) => result is Ok<T>

        Type guard that checks if a Result is successful.

        Example

        const result = ok(42);
        if (isOk(result)) {
          console.log(result.value); // TypeScript knows this is number: 42
        }
      • isErr: <T, E>(result: Result<T, E>) => result is Err<E>

        Type guard that checks if a Result is an error.

        Example

        const result = err("failed");
        if (isErr(result)) {
          console.log(result.error); // TypeScript knows this is the error: "failed"
        }
      • unwrap: <T, E>(result: Result<T, E>) => T

        Extracts the value from a successful Result or throws an error. Preserves the original error type when possible.

        Example

        const success = ok(42);
        console.log(unwrap(success)); // 42

        const failure = err(new Error("failed"));
        unwrap(failure); // Throws the original Error

        Throws

        The original error if Result is Err

      • unwrapOr: <T, E>(result: Result<T, E>, defaultValue: T) => T

        Extracts the value from a Result or returns a default value.

        Example

        const success = ok(42);
        console.log(unwrapOr(success, 0)); // 42

        const failure = err("failed");
        console.log(unwrapOr(failure, 0)); // 0
      • handle: <T>(fn: () => T) => Result<T, string>

        Safely executes a function, catching any thrown errors and converting them to a Result.

        Example

        const jsonString = '{"name": "John"}';
        const result = handle(() => JSON.parse(jsonString));
        // Returns: Ok(parsed object)

        const failed = handle(() => JSON.parse('invalid json'));
        // Returns: Err("Unexpected token i in JSON at position 0")
      • handleAsync: <T>(fn: () => Promise<T>) => Promise<Result<T, string>>

        Safely executes an async function, catching any thrown errors and converting them to a Result.

        Example

        const result = await handleAsync(async () => {
          const response = await fetch('/api/users');
          return response.json();
        });
        // Returns: Ok(users) or Err("Network error")

        • pipe: <T, E>(

          // Stops at first error, no overhead

        • all: <T, E>(results: Result<T, E>[]) => Result<T[], E>

          Converts an array of Results into a Result of array. Fails fast on the first error encountered.

          Example

          const results = [ok(1), ok(2), ok(3)];
          const combined = batch.all(results);
          // Returns: Ok([1, 2, 3])

          const mixed = [ok(1), err("failed"), ok(3)];
          const failed = batch.all(mixed);
          // Returns: Err("failed")
        • allAsync: <T, E>(promises: Promise<Result<T, E>>[]) => Promise<Result<T[], E>>

          Waits for all Promise to settle, then combines like batch.all().

          Example

          const promises = [fetchUser(1), fetchUser(2), fetchUser(3)];
          const result = await batch.allAsync(promises);
          // Returns: Ok([user1, user2, user3]) or first Err
        • allSettledAsync: <T, E>(promises: Promise<Result<T, E>>[]) => Promise<{ oks: T[]; errors: E[] }>

          Waits for all Promise to settle and partitions successes from errors. Unlike allAsync, this never fails and returns both successes and errors.

          Example

          const promises = [fetchUser(1), fetchUser(2), fetchUser(3)];
          const { oks, errors } = await batch.allSettledAsync(promises);
          console.log(`Loaded ${oks.length} users, ${errors.length} failed`);
        • oks: <T, E>(results: Result<T, E>[]) => T[]

          Extracts all success values from an array of Results. Performance: ~2-3x faster than filter().map() chains.

          Example

          const results = [ok(1), err("failed"), ok(3), err("error")];
          const successes = batch.oks(results);
          // Returns: [1, 3]
        • errs: <T, E>(results: Result<T, E>[]) => E[]

          Extracts all error values from an array of Results. Performance: ~2-3x faster than filter().map() chains.

          Example

          const results = [ok(1), err("failed"), ok(3), err("error")];
          const errors = batch.errs(results);
          // Returns: ["failed", "error"]
        • partition: <T, E>(results: Result<T, E>[]) => { oks: T[]; errors: E[] }

          Separates an array of Results into successes and errors.
