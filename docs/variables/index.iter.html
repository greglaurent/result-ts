    • map: <T, U, E>(result: Result<T, E>, mapper: (value: T) => U) => Result<U, E>

      Transforms the success value of a Result using the provided function.

      Example

      const result = ok(5);
      const doubled = iter.map(result, x => x * 2);
      // Returns: Ok(10)

      const error = err("failed");
      const mapped = iter.map(error, x => x * 2);
      // Returns: Err("failed") - unchanged

      Transforms the success value of a Promise using an async function.

      High-performance operation chaining with early exit on first error. Lightweight alternative to generators for simple sequential operations.

      Performance: ~85% faster than generator-based approaches for simple chains.

      Example

      const result = iter.pipe(
        getUser(1),
        user => validateUser(user),
        user => enrichUser(user),
        user => saveUser(user)
